# Тест для тебя 💖

Милый интерактивный сайт-тест. Она проходит тест — её ответы **отправляются на сервер**, а ты заходишь в защищённую паролем админку и видишь все её попытки: сколько раз проходила и какие ответы были в каждой.

## Как это работает

```
Она:  index.html  →  отвечает на вопросы  →  ответы летят на сервер (save.php)
                                                       │
                                                       ▼
                                          data/attempts.json (все попытки)
                                                       │
Ты:   admin.php  ←  вводишь пароль  ←──────────────────┘
```

- Каждое прохождение теста = **отдельная попытка** со своей датой и временем. Попытки не накладываются друг на друга, даже если пройдены в разное время.
- Данные хранятся на сервере в файле `data/attempts.json` (база данных не нужна).

## 1 часть — тест (`index.html`)

- Первый вопрос: «Ты меня любишь?» с кнопками **Да** и **Нет**. Кнопку **Нет** нажать нельзя — она убегает 😏
- Ещё 5 вопросов с вариантами ответов:
  1. Что ты почувствовала, когда мы познакомились?
  2. Чем тебе больше всего нравится заниматься вместе?
  3. Как сильно ты по мне скучаешь?
  4. Куда бы ты хотела отправиться со мной?
  5. Сколько сердечек я заслужил?
- Прогресс-бар показывает, сколько пройдено. В конце ответы отправляются на сервер.

## 2 часть — админка (`admin.php`)

- Открываешь `admin.php`, вводишь **пароль**.
- Видишь все попытки (новые сверху): номер попытки, дату/время и все ответы.
- Сверху — счётчик общего числа попыток.
- Кнопки «Очистить всё» и «Выйти».

## ⚙️ Настройка перед загрузкой на сервер

1. Открой `config.php` и **поменяй пароль**:
   ```php
   $ADMIN_PASSWORD = 'свой_надёжный_пароль';
   ```
2. Залей все файлы на хостинг с поддержкой **PHP** (Beget, Timeweb, reg.ru, любой shared-хостинг и т.п.) — обычно по FTP в папку сайта.
3. Убедись, что папка `data/` доступна для записи (права `755` или `775`). Файл `data/attempts.json` создастся сам при первой попытке.
4. Открой сайт — отдай ей ссылку на `index.html` (или просто на корень сайта), а себе оставь `admin.php`.

## 🖥️ Установка на сервер Debian (заход по IP)

Пошагово, как поднять сайт на чистом сервере Debian (11/12) и открывать его по IP-адресу, например `http://123.45.67.89`.

### 1. Подключись к серверу по SSH

```bash
ssh root@123.45.67.89
```
(вместо `123.45.67.89` — IP твоего сервера)

### 2. Установи веб-сервер Apache и PHP

```bash
apt update
apt install -y apache2 php libapache2-mod-php
```

### 3. Залей файлы сайта

Корень сайта по умолчанию — `/var/www/html`. Удали стандартную страницу и положи туда файлы проекта:

```bash
rm -f /var/www/html/index.html
```

Загрузить файлы можно одним из способов:

- **Через git** (если проект в репозитории):
  ```bash
  apt install -y git
  rm -rf /var/www/html && git clone <ссылка_на_репозиторий> /var/www/html
  ```
- **Через scp** со своего компьютера (выполнять локально, не на сервере):
  ```bash
  scp -r ./* root@123.45.67.89:/var/www/html/
  ```

### 4. Поставь права на папку с данными

Веб-серверу нужно право записи в `data/`, чтобы сохранять попытки:

```bash
chown -R www-data:www-data /var/www/html
chmod -R 755 /var/www/html
chmod -R 775 /var/www/html/data
```

### 5. Включи поддержку .htaccess (чтобы папка data была закрыта)

```bash
a2enmod rewrite
```

Открой конфиг `/etc/apache2/apache2.conf` и убедись, что для `/var/www/` стоит `AllowOverride All`:

```apache
<Directory /var/www/>
    Options Indexes FollowSymLinks
    AllowOverride All
    Require all granted
</Directory>
```

Перезапусти Apache:

```bash
systemctl restart apache2
```

### 6. Поменяй пароль админки

```bash
nano /var/www/html/config.php
```
Замени `changeme` на свой пароль, сохрани (Ctrl+O, Enter, Ctrl+X).

### 7. Открой порт 80 (если включён фаервол)

```bash
ufw allow 80/tcp
```

### 8. Готово — заходи по IP

- Тест (для неё):  `http://123.45.67.89/`
- Админка (для тебя): `http://123.45.67.89/admin.php`

> Если хочешь красивый адрес вместо IP — нужен домен; скажи, добавлю инструкцию.

## 🔒 Безопасность

- Пароль админки лежит в `config.php` — **обязательно поменяй** значение по умолчанию `changeme`.
- Папка `data/` закрыта от прямого доступа через браузер файлом `data/.htaccess` (для Apache). Если у тебя Nginx — закрой доступ к `/data/` в конфиге сервера.
- Реальные ответы (`data/attempts.json`) не попадают в репозиторий — см. `.gitignore`.

## Файлы

| Файл | Назначение |
|------|------------|
| `index.html` | Страница с тестом |
| `script.js` | Логика теста: вопросы, убегающая кнопка, отправка ответов, анимации |
| `hearts.js` | Плавающие сердечки на фоне (фон теста и админки) |
| `style.css` | Современные стили теста и админки |
| `save.php` | Принимает ответы и сохраняет попытку в `data/attempts.json` |
| `admin.php` | Админ-страница с паролем: показывает все попытки |
| `config.php` | Настройки: пароль админки и путь к файлу данных |
| `data/` | Хранилище попыток (закрыто от прямого доступа) |

## Локальный запуск (для проверки)

Нужен PHP:

```bash
php -S localhost:8000
# тест:    http://localhost:8000/index.html
# админка: http://localhost:8000/admin.php
```
