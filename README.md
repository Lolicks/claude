# Тест для тебя 💖

Милый интерактивный сайт-тест: проходишь вопросы про любовь, а все попытки сохраняются и показываются на отдельной странице.

## Что внутри

**1 часть — тест (`index.html`)**
- Первый вопрос: «Ты меня любишь?» с кнопками **Да** и **Нет**. Кнопку **Нет** нажать нельзя — она убегает от курсора и пальца 😏
- Ещё 5 вопросов с вариантами ответов (без убегающей кнопки):
  1. Что ты почувствовала, когда мы познакомились?
  2. Чем тебе больше всего нравится заниматься вместе?
  3. Как сильно ты по мне скучаешь?
  4. Куда бы ты хотела отправиться со мной?
  5. Сколько сердечек я заслужил?
- Прогресс-бар показывает, сколько вопросов пройдено.

**2 часть — результаты (`results.html`)**
- Показывает **все попытки** прохождения теста.
- Каждая попытка записывается отдельно со своей датой и временем, поэтому ответы, данные в разное время, **не накладываются** друг на друга.
- Новые попытки отображаются сверху.
- Кнопка «Очистить всё» удаляет историю попыток.

## Как хранятся ответы

Ответы сохраняются в браузере через `localStorage` (ключ `loveQuizAttempts`).
Каждая попытка — это отдельная запись:

```json
{
  "id": 1730000000000,
  "date": "2026-05-30T17:00:00.000Z",
  "answers": [
    { "question": "Ты меня любишь? 💖", "answer": "Да 💕" }
  ]
}
```

Сервер не нужен — всё работает прямо в браузере.

## Как запустить

Просто открой `index.html` в браузере. Либо подними локальный сервер:

```bash
python3 -m http.server 8000
# затем открой http://localhost:8000
```

| Файл | Назначение |
|------|------------|
| `index.html` | Страница с тестом (1 часть) |
